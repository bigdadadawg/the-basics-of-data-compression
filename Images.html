The Basics of Data Compression

Images

An image is a binary representation of visual information such as drawings, pictures, graphs, logos, or individual video frames[6]. Image compression is minimizing the size in bytes of a graphics file[7]. Compression can be either lossless or lossy. Lossless compression retains all information whereas lossy compression intentionally discards some information.

The Difference Between Lossless and Lossy Compression on Images.

With lossless compression, every single bit of data that was originally in the file remains after the file is uncompressed. All of the information is completely restored. Lossy compression reduces a file by permanently eliminating certain information, especially redundant information[8].

Why Is It Important For Web Designers To Compress Image Files?

There are many reasons why a web designer would want to compress image files, these can include:

  • big files can make your pages load too slowly[9]
  • search engines can add "load time" to their algorithms, this means that if a site takes too long to load, they will be penalised[9]
  • people with dial-up would not be able to easily load them[9]

The Effects of Lossless and Lossy Compression

Both lossless and lossy compression can result in the loss of quality, while both can significantly reduce the file size.
