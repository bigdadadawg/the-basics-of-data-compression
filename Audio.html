The Basics of Data Compression

Audio

Audio is sound within the acoustic range available to humans. An audio file is a record of captured sound that can be played back[1]. Audio compression is the process of lessening the dynamic range between the loudest and quietest parts of an audio signal[2]. Compression can be either lossless or lossy. Lossless compression retains all information whereas lossy compression intentionally discards some information.

Lossless Audio Files

Examples of Lossless File Types

  • FLAC[3]

  • WAV[3]

  • ALAC[3]

  • WMA Lossless[3]

Why Use Lossless File Formats?

Lossless file formats are ideal when it is necessary for the uncompressed file to be returned to it's original state.

Lossy Audio Files

Examples of Lossy File Types

  • MP3[4]

  • WMA[4]

  • AAC[4]

  • Ogg Vorbis[4]

How Does Lossy Compression Work?

Lossy compression makes certain assumptions about frequencies that the human ear is unlikely to detect. When a song for example is converted to a lossy audio format such as AAC, the algorithm analyses all the frequencies. It then discards ones that the human ear shouldn't be able to detect[5].

Why Use Lossy File Formats?

Lossy file formats are ideal when memory space conservation is necessary[10].
